This collection includes scripts for

(1) preprocessing csv-style data into the input the compatibility code
scripts need

(2) running the compatibility code analysis for the HV, Hayes, and OT
representations described in Pearl, Ho, & Detrano 2014.

Pearl, L., Ho, T., & Detrano, Z. 2014. More learnable than thou?
Testing metrical phonology representations with child-directed
speech. Proceedings of the Berkeley Linguistics Society, 398-422.

(3) Analyzing the output of the compatibility code analysis.

Specific instructions for all these components are included in the
ReadMe.txt files in each of the subfolders: data-preprocessing,
parametric, OT.
